---
title: "RESUME"
description: "Triggered when a paused timer is resumed."
---

## Usage Example

```java
@EventHandler
public void onVCTEvent(VCTEvent event) {
    if (event.getType() == VCTEventType.RESUME) {
        Bukkit.broadcastMessage("The timer " + event.getTimerId() + " has resumed!");
    }
}
```

## Behavior

* Fired when a paused timer continues counting down.
