## Requirements

#### 1.- Minecraft 1.13\+

#### 2.- Spigot, Paper or Purpur

<Warning>
  You need Spigot, Paper, or Purpur for the plugin to work. And don't use Craftbukkit.
</Warning>

#### 3.- PlaceholderAPI

<Info>
  This is an optional dependency, but recommended.

  https://www.spigotmc.org/resources/placeholderapi.6245/
</Info>

## Installation

<Steps>
  <Step title="Download the plugin">
    Download the latest version of VCT from [Modrinth](https://modrinth.com/datapack/voiid-countdown-timer) OR you can view the other official sites [here.](/get-started/quick-overview#where-can-vct-be-downloaded%3F)
  </Step>
  <Step title="Place the plugin in the plugins folder">
    Find the plugins folder on your server and place the plugin in that folder.
  </Step>
  <Step title="Restart">
    Restart the server.
  </Step>
  <Step title="Final checking">
    Check that a message similar to this has appeared on the console:

    ![installed.png](/images/installation-or-updation/installed.png)

    If so, it means that you have successfully installed VCT\!
  </Step>
</Steps>

## Updation

<Steps>
  <Step title="Download the plugin">
    Download the latest version of VCT from [Modrinth](https://modrinth.com/datapack/voiid-countdown-timer) OR you can view the other official sites [here.](/get-started/quick-overview#where-can-vct-be-downloaded%3F)
  </Step>
  <Step title="Deleting">
    Remove the old plugin file, usually called `VoiidCountdownTimer.jar`.

    Additionally, it is recommended to delete the VoiidCountdownTimer folder in the plugins folder to obtain the new features in `config.yml`, but this is not mandatory.
  </Step>
  <Step title="Place the plugin in the plugins folder">
    Find the plugins folder on your server and place the plugin in that folder.
  </Step>
  <Step title="Restart">
    Restart the server.
  </Step>
  <Step title="Final checking">
    Check that a message similar to this has appeared on the console:

    ![installed.png](/images/installation-or-updation/installed.png)

    If so, it means that you have successfully installed VCT\!
  </Step>
</Steps>
